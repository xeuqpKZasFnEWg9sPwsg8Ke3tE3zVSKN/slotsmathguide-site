# UX QA Checklist (v2 release gate)

Goal: ship a site-wide quality jump, not incremental tweaks.

## Visual hierarchy
- [ ] Homepage hero reads as premium (headline, lead, CTAs, sections)
- [ ] Hubs have clear sections + scannable cards
- [ ] Article pages have clear intro, takeaways, and sections

## Typography & spacing
- [ ] Line length comfortable (no full-width paragraphs)
- [ ] Headings scale correctly
- [ ] Consistent spacing between sections

## Components
- [ ] Buttons consistent and feel clickable
- [ ] Cards consistent across home/hubs
- [ ] Callouts used sparingly and consistently
- [ ] Tables readable on mobile (scroll wrapper)

## Navigation
- [ ] Sticky header works without jank
- [ ] Active nav state highlights correctly
- [ ] Footer contains trust + policy links

## Mobile polish
- [ ] Tap targets large enough
- [ ] No horizontal overflow
- [ ] Hero and cards stack cleanly

## Trust & professionalism
- [ ] About/Terms/Privacy/Disclosure/Contact accessible
- [ ] Ontario scope messaging present where needed

## Performance
- [ ] No heavy JS
- [ ] CSS is one file
- [ ] Pages load fast
